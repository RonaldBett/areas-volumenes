Introducción

Como parte de la competencia 240201528 (Razonar cuantitativamente frente a situaciones susceptibles de ser abordadas de manera matemática), del programa de formación de Análisis y desarrollo de software ADSO, se busca conocer, interiorizar e implementar el cálculo de áreas, perímetros y volúmenes de figuras geométricas. Es por ello que se realizó un algoritmo que permite calcular el área y perímetro de 5 figuras planas; el área y volumen de 5 sólidos regulares; y el volumen de figuras irregulares.
En este caso se escribió el algoritmo utilizando JavaScript, y para interactuar con él se utiliza HTML y CSS.

Área y Volumen

Para calcular Área y Volumen seleccione alguna de las figuras disponibles:

Para calcular el volumen de un sólido irregular, podemos usar el principio de Arquímedes, que establece que todo cuerpo sumergido total o parcialmente en un fluido experimenta una fuerza de empuje vertical hacia arriba igual al peso del fluido desalojado. Esta premisa permite calcular el volumen de objetos irregulares midiendo el volumen o peso del agua que desplazan al sumergirse.

En este caso, usando un recipiente con forma cilíndrica relleno con agua, podemos calcular el volumen del agua que se desplaza hacia arriba al sumergir un cuerpo.
